New Package Roundup

August 8, 2014 izuzak Tweet

If you’ve been looking for new Atom packages to try out, here’s a few neat ones we noticed since the last package roundup post. Enjoy!

Pain Split

This package by @vidius extends the default set of commands for splitting panes so that you can create an empty file in the new pane or move the active tab into the new pane. Best package name ever.

Ask StackOverflow

Stuck working on your project or homework? This package can help! Quickly find answers and code samples from Stack Overflow directly from Atom. Great work @chris911!

Git Blame

If looking at Git blame is a part of your workflow, this handy package created by @alexcorre helps you do that right from Atom. Works with submodules, and allows one-click access to commit pages on remote repositories (GitHub, Bitbucket, or your remote of choice).

Hex

Need a hex viewer in your toolbox? @abeestrada has you covered.

Jumpy and Easy Motion

Jumpy, from @DavidLGoldberg, creates dynamic hot keys to quickly jump around files and across visible panes.

And @adrian-budau has a similar package, easy-motion.

TODO Show

This package from @jamischarles finds all TODO, FIXME and CHANGED comments in your project and shows them nicely organized in a pane. You can also configure the package to find other types of comments.

Seti Theme: Seti UI and Seti Syntax

This dark-colored theme, created by @jesseweed, has been one of our favorites for a while. It ships with custom file icons and a great syntax color scheme, so make sure you give it a try.
